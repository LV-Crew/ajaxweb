SigmaVisual Overview

When you first start SigmaVisual, you are presented with the GUI builder with a new page created.  Generally, a page has three views, that is , normal view, structure view and design view residenting in a tab control. See following figure.

[Design View] is what you see at first sight. It is mainly made of three parts, [Tool Box], [Design Area] and [Component Config Window].
  • [Tool Box] - Hoding many components for you to choosing.
  • [Design Area] - To drag widgets from [Tool Box] here and set its properties. That is what you need to do only.
  • [Component Config Window] - If one component is selected, its properties and events will appear in this window. You could view its current values or change them to different ones.
